# Adding your own


This is an how you can add your own [bgcolor=#343942][code]SectionResources[/code][/bgcolor] and [bgcolor=#343942][code]ItemResources[/code][/bgcolor] to the addon.
Have fun tinkering!


## Hierarchy
[ul][bgcolor=#343942][code]sections/section_name/[/code][/bgcolor] contains [b]one[/b] [bgcolor=#343942][code]SectionResource[/code][/bgcolor], its [bgcolor=#343942][code]ItemResources[/code][/bgcolor] and their [bgcolor=#343942][code]pictures[/code][/bgcolor].[/ul]

[ul][bgcolor=#343942][code]items/[/code][/bgcolor] contains every [bgcolor=#343942][code]ItemResource[/code][/bgcolor].[/ul]

[ul][bgcolor=#343942][code]pictures/[/code][/bgcolor] contains every picture. [/ul]

[indent][img]pictures/hierarchy.PNG[/img][/indent]


## Adding a new [bgcolor=#343942][code]Section[/code][/bgcolor]

1. According to the above hierarchy, create a new folder [bgcolor=#343942][code]sections/new_section/[/code][/bgcolor]
2. Right-click the folder -> Choose [b]Create New[/b] -> Choose [b]Resource[/b]
3. Search for [bgcolor=#343942][code]SectionResource[/code][/bgcolor], press [b]Create[/b] and give a file name
4. Double-click on the newly created [bgcolor=#343942][code]SectionResource[/code][/bgcolor] -> Its properties appear in the [b]Inspector[/b] tab.
5. Change the [b]Name[/b] & add your [bgcolor=#343942][code]ItemResources[/code][/bgcolor]
6. Finally, add the [bgcolor=#343942][code]SectionResource[/code][/bgcolor] to the addon's main scene
[indent][ul]Open [bgcolor=#343942][code]godot_style/main.tscn[/code][/bgcolor][/ul][/indent]
[indent][ul]In the [b]Scene[/b] tab, double-click [b]Main[/b], which is a [b]Control[/b] node[/ul][/indent]
[indent][ul]In the [b]Inspector[/b] tab, add your [bgcolor=#343942][code]SectionResource[/code][/bgcolor] to the array [b]Sections[/b][/ul][/indent]

[indent][indent][img]pictures/main.PNG[/img][/indent][/indent]

>[b]Note:[/b] Disable and re-enable the addon to trigger any updates (or reload the current project or the game engine)



## Adding a new [bgcolor=#343942][code]Item[/code][/bgcolor]

1. According to the above hierarchy, create a new folder [bgcolor=#343942][code]sections/any_section/items[/code][/bgcolor]
2. For every new [b]Item[/b]:
[indent][ul]Right-click the folder -> Choose [b]Create New[/b] -> Choose [b]Resource[/b][/ul][/indent]
[indent][ul]Search for [bgcolor=#343942][code]ItemResource[/code][/bgcolor], press [b]Create[/b] and give a file name[/ul][/indent]
[indent][ul]Double-click on the newly created [bgcolor=#343942][code]ItemResource[/code][/bgcolor] -> Its properties appear in the [b]Inspector[/b] tab.[/ul][/indent]
[indent][ul]Change the [b]Name[/b] & add your [b]Pictures[/b][/ul][/indent]
4. Finally, add the [bgcolor=#343942][code]ItemResources[/code][/bgcolor] to their corresponding [bgcolor=#343942][code]SectionResource[/code][/bgcolor]
[indent][img]pictures/section_resource.PNG[/img][/indent]

>[b]Note:[/b] Disable and re-enable the addon to trigger any updates (or reload the current project or the game engine)
